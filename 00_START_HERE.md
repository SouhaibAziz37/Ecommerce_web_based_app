## 🔑 Quick Reference

### File Locations
```
index.html            ← Main page
css/style.css         ← All styling
js/script.js          ← All functionality
```

### Start Website
```bash
# Double-click index.html, OR
# Python: python3 -m http.server 8000
# Then open: http://localhost:8000
```

### Key Information
- **Products**: 30 items (10 each category)
- **Price Range**: Rs. 1,800 - Rs. 12,000
- **Colors**: Blue (#6c63ff), Red (#ff6b6b), Teal (#4ecdc4)
- **Responsive**: Mobile (< 480px), Tablet (768px), Desktop (1200px+)

### Common Tasks
- **Change color**: Edit `css/style.css` `:root` variables
- **Add product**: Edit products array in `js/script.js`
- **Change text**: Find & replace in `index.html`
- **Contact info**: Search in `index.html` and update
- **Deploy**: See [SETUP.md → Deployment](SETUP.md#deployment)
